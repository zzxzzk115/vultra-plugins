# Vultra Plugins

Central catalog for Vultra plugins.

The root `plugins.json` file is the machine-readable plugin list. Editors and command-line tooling can fetch it directly, enumerate `plugins[]`, then install a selected entry from its `source` descriptor.

Default catalog URL:

```text
https://raw.githubusercontent.com/zzxzzk115/vultra-plugins/main/plugins.json
```
